# Multivariable Calculus Visualization in MATLAB

## Overview

I did this project in Calculus III using MATLAB to visualize and analyze several concepts from the course, including planes, curves in three dimensional space, velocity vectors, normal vectors, tangent planes, and surfaces.

The goal was to connect symbolic and geometric reasoning with computational visualization. Instead of treating multivariable calculus as only algebraic manipulation, the project emphasized how MATLAB can make three dimensional relationships easier to interpret.

This public version summarizes the modeling and visualization approach while omitting course specific prompts, assignment structure, and full solution code. - Tom

## Visualization Approach

I used MATLAB here to create three dimensional plots of mathematical objects and relationships.

Plane and surface problems were visualized using surface plots, axis labels, grid controls, and viewpoint adjustments. I added normal vectors to show geometric orientation and orthogonality.

Vector valued functions were used to model motion through three dimensional space. The curves were plotted with start and end points, and velocity or tangent vectors were used to interpret direction and change over time.

Tangent plane and surface problems were used to help connect local linear approximation with the geometry of surfaces in three dimensions.

## Key Takeaways

For me, this project reinforced that visualization is a powerful tool for understanding multivariable calculus. Three dimensional plots made it easier to interpret planes, surfaces, curves, and vectors in ways that are difficult to see from equations alone.

The work also showed the importance of choosing appropriate views, axis scaling, and labels. A correct plot can still be hard to interpret if the visualization is not designed clearly.

The combination of computation with written mathematical explanation helped me connect MATLAB output to the underlying calculus concepts.
